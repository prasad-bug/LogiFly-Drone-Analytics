## 🚀 Quick Start

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Run the Jupyter Notebook
```bash
jupyter notebook logifly_case_study.ipynb
```
Run all cells top to bottom (Kernel → Restart & Run All)

### Step 3 — Launch the Streamlit Dashboard
```bash
streamlit run app.py
```
Opens at → http://localhost:8501

- **Annual Savings:** ₹52.5 Lakhs (35% of ₹1.5 Crore)

- **5-Year Cumulative Profit:** ~₹1.6 Crores
- **Accuracy Improvement:** 78% → 96% (+18 percentage points)
- **Error Rate Reduction:** 12% → 3% (−75%)
- **Throughput Increase:** 1,800 → 4,500 items/day (2.5×)
- **Verdict:** ✅ **YES — INVEST in Smart Logistics Drones**
